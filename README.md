# 招聘可视化分析系统 python691

## 项目概述

招聘可视化分析系统是一个基于 Python3 和 Django 框架开发的，旨在通过数据爬取、处理和可视化展示，帮助用户更好地理解招聘市场趋势和需求的系统。

## 技术栈

- Python 3
- Django 框架
- MySQL 数据库

## 功能模块

### 首页数据展示

- 用户创建时间和信息展示
- 数据信息展示
- 爬取数据时间展示
- 可视化界面

### 可视化分析

#### 薪资情况

- 通过学历筛选查看不同岗位薪资
- 使用柱状图和饼图展示薪资分布和比例

#### 企业情况

- 通过职位筛选查看主要招聘城市
- 查看公司规模和职位在各城市的分布比例

#### 学历情况

- 查看各职位对学历和工作经验的要求
- 使用漏斗图和矩形树图进行可视化展示

## 系统角色

- 用户：查看和分析招聘数据
- 管理员：负责后台管理和数据更新

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x 或更高版本

## 部署步骤

1. 安装 Python 3 和 Django
2. 配置 MySQL 数据库
3. 导入系统所需的 SQL 数据
4. 运行 Django 项目
